# NetIds - Lightweight Network Intrusion Detection System

NetIDS is a Python- based Network Intrusion Detection System (IDS) that supports 
-Live Packet sniffing
-PCAP analysis
It detects common reconnaissance and attack behavior using rolling time windows and threshold-based logic

# Features
- TCP Port Scan Detection
- TCP SYN Burst/ SYN Flood Detection
- ICMP Sweep (Ping Sweep) Detectoin
- Live Packet capture using Scapy
- PCAP file analysis
- Alert logged in JSON Lines format.
