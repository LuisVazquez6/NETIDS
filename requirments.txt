## System Requirements

- Python 3.8+
- Linux (Ubuntu / Kali recommended)
- Root privileges (required for live packet capture)
- Ollama with llama3.2:3b pulled (https://ollama.com)

## Virtual Environment Setup

# Create
python3 -m venv .venv

# Activate
source .venv/bin/activate

# Upgrade pip
pip install --upgrade pip

# Install dependencies
pip install -r requirments.txt

## Ollama Setup

# Install Ollama
curl -fsSL https://ollama.com/install.sh | sh

# Pull the model
ollama pull llama3.2:3b

## Running NetIDS

# Terminal 1 — AI backend
ollama serve

# Terminal 2 — IDS engine
sudo -E .venv/bin/python src/ids.py --live --iface enp0s3

# Terminal 3 — Dashboard
.venv/bin/streamlit run src/dashboard/app.py

## Replay a PCAP file (no root required)
.venv/bin/python src/ids.py --pcap capture.pcap

## Run Demo Attack (from Kali attacker machine)
sudo bash demo_attack.sh <victim-ip>
